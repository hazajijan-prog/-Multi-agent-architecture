# Multi-Agent Travel System

This project implements a simplified multi-agent architecture using:

- Planner
- Doer
- Critic
- Human-in-the-Loop

## Purpose

The goal is to demonstrate how agents interact and how control flows between them.

## Flow

User → Input validation → Planner → Doer → Critic → Human approval → Output

## Notes

- Functions are stubbed
- Data is hardcoded
- Focus is on architecture, not production AI
